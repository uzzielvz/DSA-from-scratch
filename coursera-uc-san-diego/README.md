## Specialization Structure

### Course 1: Algorithmic Toolbox
**Topics:** Algorithm design techniques, greedy algorithms, divide & conquer, dynamic programming basics  
**Folder:** `course1-algorithmic-toolbox/`

### Course 2: Data Structures
**Topics:** Basic data structures, dynamic arrays, priority queues, hash tables, binary search trees  
**Folder:** `course2-data-structures/`

### Course 3: Algorithms on Graphs
**Topics:** Graph traversal, shortest paths, minimum spanning trees  
**Folder:** `course3-graphs/`

### Course 4: Algorithms on Strings
**Topics:** String algorithms, pattern matching, tries, suffix trees  
**Folder:** `course4-strings/`

### Course 5: Advanced Algorithms and Complexity
**Topics:** Network flows, linear programming, NP-completeness  
**Folder:** `course5-advanced/`

### Course 6: Genome Assembly Programming Challenge
**Topics:** Applying algorithms to real-world problems  
**Folder:** `course6-genome-assembly/`
